# WorkBuddy 入门：从零开始认识这只会干活的 AI

你可能已经用过 ChatGPT、文心一言、通义千问。

它们都能聊天、能写文案、能回答问题。但你有没有这种感觉——**聊完之后，活儿还是得自己干**？

你说"帮我整理一份 Excel 报表"，它给你列了一堆步骤和公式，然后你打开 Excel，一行一行手动操作。

WorkBuddy 不一样。它不是告诉你怎么做，而是**直接帮你做**。

## 一、WorkBuddy 到底是什么

一句话定义：

> WorkBuddy 是一个能够操控你电脑、调用各种工具、自主完成多步骤任务的 AI 智能体（AI Agent）。

关键词是**智能体**，不是"聊天机器人"。

| 对比项 | 传统 AI 聊天工具 | WorkBuddy |
|--------|------------------|-----------|
| 交互方式 | 一问一答 | 下达任务，自主执行 |
| 输出形式 | 文字回复 | 实际操作 + 交付物 |
| 能否操作文件 | 不能 | 能读写、创建文件 |
| 能否运行代码 | 不能 | 能在沙箱里运行 |
| 能否联网 | 部分能 | 能搜索、能抓取网页 |
| 多步骤任务 | 需要你一步步引导 | 自主拆解、连续执行 |

简单说：传统 AI 是你的**顾问**，WorkBuddy 是你的**员工**。

## 二、它能做什么

WorkBuddy 的能力覆盖范围非常广。下面这些是它最擅长的几类活儿：

### 1. 写文档、做报告
你给它一份原始数据或一个主题，它能直接生成完整的 Word、PDF、Markdown 文档，甚至 PPT。

### 2. 处理数据
Excel 表格清洗、数据分析、生成图表、做可视化报表——它能调用 Python + Pandas + Matplotlib 直接处理，不用你手动建公式。

### 3. 开发软件
从写一个函数到搭建一个完整网站，它能读代码、写代码、跑测试、部署上线。你现在看的这个博客网站，就是 WorkBuddy 帮忙开发的。

### 4. 联网调研
需要查某个行业的最新数据？它能搜索网页、抓取内容、汇总成调研报告，附上信息来源。

### 5. 日常办公
发邮件、整理笔记、安排日程、生成会议纪要……只要能在电脑上完成的，它基本都能帮。

## 三、为什么值得用

你可能会问：我自己手动也能干这些事，为什么要用 WorkBuddy？

三个理由：

**1. 速度**
一篇 3000 字的行业调研报告，你手动查资料、整理、排版，至少要半天。WorkBuddy 十几分钟就能交付初稿。

**2. 门槛低**
你不需要会 Python 才能做数据分析，不需要会 React 才能搭网站。你只需要用大白话描述你想要什么，WorkBuddy 帮你搞定技术细节。

**3. 可迭代**
第一版不满意？直接说"把图表换成柱状图""第二段太啰嗦，精简一下"，它会基于已有成果修改，不用从头来。

## 四、怎么开始用

### 第 1 步：打开 WorkBuddy
在支持的平台上启动 WorkBuddy，你会看到一个类似对话框的界面。但记住——这不是聊天框，是**任务下达框**。

### 第 2 步：下达第一个任务
别一上来就写复杂指令。先从简单的开始，建立感觉。比如：

```
帮我写一篇 500 字的短文，主题是"为什么早起能提升工作效率"，
风格轻松口语化，结尾加一句金句。
```

### 第 3 步：观察它的工作过程
WorkBuddy 执行任务时，你会看到它在"思考"——它会拆解步骤、调用工具、读取文件、运行代码。这个过程是透明的，你能看到它每一步在干什么。

### 第 4 步：给反馈，让它改进
第一版出来后，直接提修改意见：

```
开头太正式了，改得像跟朋友聊天一样。
字数压缩到 400 字。
```

它会基于上一版调整，而不是重写。

## 五、新手最容易犯的 3 个错误

### 错误 1：把它当聊天机器人
❌ "你好""在吗""你觉得 AI 会取代人类吗"
✅ 直接给任务："帮我写一份周报，本周完成了 A、B、C 三件事，下周计划做 D 和 E"

### 错误 2：指令太模糊
❌ "帮我做个报表"
✅ "把这个 Excel 文件按销售额降序排列，取前 10 名，做成柱状图，导出成 PNG"

### 错误 3：一次塞太多任务
❌ "帮我写文章、做封面图、建网站、发邮件"
✅ 一次一个任务，做完再说下一个。复杂任务可以让它分步执行，但别一股脑全丢给它。

## 六、写在最后

WorkBuddy 的核心价值就一句话：**把你的意图变成结果，中间的过程它自己搞定**。

你不需要学编程、不需要学设计、不需要会写复杂公式。你只需要知道"我想要什么"，然后把它说清楚。

剩下的，交给 WorkBuddy。
