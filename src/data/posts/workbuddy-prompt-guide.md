# WorkBuddy 指令填写完全指南：从"聊天"到"干活"只差一个 Prompt

很多人装了 WorkBuddy，用了三天就放一边了。

不是工具不行，是指令没写对。

WorkBuddy 不是一个聊天机器人——它是一只能操控你电脑的 AI 智能体。你跟它说的话，决定了它是帮你干活，还是给你讲废话。

这篇文章把 WorkBuddy 的指令填写技巧一次讲透。

## 一、核心公式：动作 + 对象 + 要求 + 交付

写好 WorkBuddy 指令，记住这个万能公式：

```
【动作】+【对象】+【具体要求】+【交付物格式】
```

对比一下：

| ❌ 错误写法 | ✅ 正确写法 |
|-----------|-----------|
| "帮我整理一下文件" | "扫描**桌面**，将**所有 .xlsx 文件**按创建时间**分类到 3 个文件夹**：本周/本月/更早，完成后**告诉我一共整理了多少个文件**" |
| "分析一下这份数据" | "打开 **D:\数据\销售.xlsx**，删除重复行，按**地区**汇总销售额，生成**柱状图**，输出为 **销售分析_已处理.xlsx**" |
| "写个周报" | "把**本周工作碎片**整合成周报，按**成果+问题+计划**三块写，每项**附带数据**，语气**正式专业**" |

> 模糊的指令 = 模糊的结果。WorkBuddy 不会读心，你给它多少信息，它就还你多少价值。

## 二、5 套万能 Prompt 公式（直接套用）

### 公式 1：角色 + 任务 + 格式

```
我是【角色】，请帮我【任务】，输出格式为【格式要求】
```

**示例：**
```
我是产品经理，请帮我把近 30 天用户反馈按功能模块分类，
标注正面/负面情绪，提取 Top5 紧急问题，
输出为优先级排序表格。
```

### 公式 2：角色 + 目标 + 约束

```
你现在是【专业角色】，目标是【具体目标】，
要求：1. xxx  2. xxx  3. xxx
```

**示例：**
```
你现在是互联网行业分析师，目标是写一份 2026 AI 办公工具赛道分析报告，
要求：
1. 字数 1500 字以内
2. 覆盖 3 个以上竞品
3. 每个竞品用"产品名-核心功能-差异化优势"三段式结构
4. 最后给出 3 条投资建议
```

### 公式 3：输入 + 处理规则 + 输出

```
输入数据是【数据描述】，请按以下规则处理：
【规则 1、规则 2、规则 3】
输出要求：【格式和字段】
```

**示例：**
```
输入数据是"Q1 销售数据.xlsx"的 Sheet1，请按以下规则处理：
1. 删除"状态"列为"已取消"的行
2. 按"区域"列分组求和
3. 按销售额从高到低排序
输出为新文件，保留原始数据备份。
```

### 公式 4：场景 + 结构 + 语气

```
针对【使用场景】，输出结构为【1/2/3 部分】，
语气【正式/轻松/专业】，目标读者是【xxx】
```

**示例：**
```
写一封产品介绍邮件，结构分为：市场规模、产品功能、竞争优势、财务预测四个板块，
语气专业但亲切，目标读者是潜在投资人，
结尾加上"期待进一步沟通"的 CTA。
```

### 公式 5：定时 + 任务 + 推送

```
【时间频率】，执行【具体操作】，将结果【推送到哪里】
```

**示例：**
```
每天早上 9 点，扫描 AI 赛道最新 5 条核心新闻，
聚焦 AI 编程和具身智能两大板块，
每条含事件详情 + 一句话行业解读，
以要点形式推送到企业微信"行业资讯群"。
```

## 三、高频场景 Prompt 模板（复制即用）

### 📄 文档处理

```
读取 D:\周报\2026Q2数据.xlsx，
生成季度复盘 PPT，包含：数据图表、风险总结、Q3 落地计划，
商务简约排版，保存至 D:\周报\归档
```

### 📊 数据处理

```
打开桌面"用户调研.csv"，
去重、填充分类缺失值，
按"年龄段"分组统计平均满意度，
生成柱状图，导出为 Excel
```

### 📝 会议纪要

```
根据以下会议录音文本，生成结构化纪要：
【议题】、【结论】、【行动项】（含负责人和截止时间）
不要编造原文不存在的信息，语气客观中立
```

### ✍️ 周报生成

```
整合我本周工作碎片（附在下方），生成周报：
第一块：本周成果（每项带数据）
第二块：遇到问题及解决方案
第三块：下周计划（分必做/弹性）
语气正式，可直接提交
```

### 🎨 内容创作

```
你是文案专家，写一篇小红书风格的种草文案：
主题：AI 办公工具推荐
目标读者：职场新人
包含：痛点引入 + 3 款工具对比 + 使用场景 + 结尾互动
标题抓眼球，正文有钩子
```

### 🔍 信息调研

```
深度调研 2026 年 AI 编程工具赛道：
1. 列出 Top5 产品及核心能力
2. 对比定价策略
3. 分析差异化优势
4. 预测下半年趋势
输出为结构化表格
```

### 🌐 网页生成

```
制作深色主题数据看板页面：
顶部 4 块核心指标卡片（新增用户、活跃率、转化率、营收）
中间搭配趋势折线图
下方增设分类柱状图
底部展示 Top5 榜单
配色蓝绿渐变，自适应手机端
```

### ⏰ 定时自动化

```
每周一上午 9 点：
1. 汇总上周所有未读消息，按紧急度排序
2. 梳理本周日程，标注时间冲突会议
3. 生成"本周核心任务"清单
推送至企业微信
```

## 四、三个关键技巧

### 1. 给"剧本"，别"问问题"

❌ 低手：*"帮我写个总结"* → 得到一篇流水账

✅ 高手：*"你是项目助理，背景是 xxx，需要总结 xxx，按 1/2/3 结构，格式为表格，语气正式"* → 得到能直接交差的成果

**区别在于：高手给了角色、背景、结构、格式——这是一套完整的"剧本"，AI 照着演就行。**

### 2. 用绝对路径，别让 AI 猜

❌ *"帮我整理一下下载文件夹"*
✅ *"扫描 C:\Users\Downloads，将 .jpg 和 .png 移动到桌面\图片备份"*

### 3. 先喂知识库，再派活

在 WorkBuddy 设置里上传你的常用模板、优秀案例、公司制度文档。喂得越细，它吐出来的东西就越像你亲手写的。

## 五、三种模式的选择策略

| 模式 | 什么时候用 | 特点 |
|------|----------|------|
| **Craft** | 任务明确，直接开干 | AI 直接操作文件、生成结果 |
| **Plan** | 复杂的多步骤任务 | AI 先出方案，你确认后再执行 |
| **Ask** | 纯咨询、方案讨论 | 只回答问题，不动任何文件 |

> 简单任务用 Craft，复杂任务用 Plan，不确定的时候用 Ask 先聊清楚。

WorkBuddy 的本质不是"AI 聊天工具"，而是一个**坐在你电脑前的 AI 助手**。

你跟它沟通的方式，决定了它是帮你省 3 小时，还是浪费你 3 分钟。

```
【动作】+【对象】+【要求】+【交付】
```
