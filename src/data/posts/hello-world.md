# 你好，世界 👋

欢迎来到我的个人博客！

这是我的第一篇文章。一直以来都想搭建一个属于自己的博客，作为一个记录生活、分享技术、沉淀思考的地方。

## 为什么要写博客？

> 写作是最好的思考方式。

写博客对我来说有几个目的：

1. **记录成长** — 技术学习的笔记、解决问题的思路，都值得被记录下来
2. **深度思考** — 把模糊的想法写清楚，本身就是一种深度思考的过程
3. **分享交流** — 或许某篇文章能帮助到别人，也或许有人愿意和我交流讨论

## 关于这个博客

这个博客使用 **React + Vite** 构建，支持 Markdown 渲染。

文章以 `.md` 文件的形式存储在项目中，使用 `gray-matter` 解析文章元数据（Front Matter）。

```js
// 文章 front matter 示例
---
title: 文章标题
date: 2025-03-15
tags: [技术, React]
summary: 文章摘要
---
```
