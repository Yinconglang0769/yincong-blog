## 图片怎么加的？

很简单，在 Markdown 里这样写：

```markdown
![图片说明文字](/images/图片文件名.svg)
```

## 支持的图片格式

- `.jpg` / `.jpeg` — 照片首选
- `.png` — 带透明背景的图
- `.gif` — 动图
- `.svg` — 矢量图，体积小不失真
- `.webp` — 现代 web 格式，体积最小
